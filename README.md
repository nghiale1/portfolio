# Portfolio — Lê Minh Nghĩa

Website portfolio tĩnh, responsive, không phụ thuộc framework hoặc thư viện ngoài.

## Chạy trên máy

Mở trực tiếp `index.html`, hoặc chạy local server:

```bash
python -m http.server 8080
```

Sau đó truy cập `http://localhost:8080`.

## Cấu trúc

- `index.html`: nội dung website
- `styles.css`: toàn bộ giao diện và responsive
- `script.js`: hiệu ứng reveal, menu mobile, tab kỹ năng, custom cursor
- `assets/le-minh-nghia.webp`: ảnh chân dung đã tối ưu cho web

## Chỉnh sửa nhanh

- Email, số điện thoại và link TikTok nằm trong `index.html`.
- Màu chủ đạo nằm ở phần `:root` trong `styles.css`.
- Có thể deploy trực tiếp lên Netlify, Vercel, GitHub Pages hoặc hosting tĩnh.
