# Visual Impact Pass

Changes made:

- Added a dedicated `HeroStage` with real vertical project posters, luminous spline trails, particles, a central content core, and responsive pointer/scroll motion.
- Kept the WebGL scene visible behind the activation screen instead of covering it with a flat opaque black layer.
